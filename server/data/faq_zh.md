# G-Fix Solution 售后服务与保修条款 (Warranty & Terms)
## 1. 保修排除范围
- **物理损坏与进水:** 
- 任何物理损坏（如碎屏、设备弯曲）和进水损坏均不在 G-Fix Solution 的保修范围内。
  - 若收到的设备已存在物理损坏，保修将不适用。
- **第三方维修:** - 未经 G-Fix Solution 事先同意，严禁进行第三方维修。擅自拆修将导致保修失效。

## 2. 维修、更换与退款政策
- **处理顺序:** 我们的目标首选是**维修**。若无法维修，我们将提供**更换**；若无法更换，将提供**全额退款**。
- **退款规则:** - 不接受因“改变主意 (Change of mind)”而发起的退款。
- 对于符合条件的非缺陷退货，可能收取 **15% 的重新上架费**。

## 3. 消费者权益法案 (New Zealand Consumer Guarantees Act)
- 我们严格遵守 1993 年《消费者保障法》。用户享有要求在合理时间内修复故障的权利。
- 如果故障无法修复或属于重大故障，用户有权要求更换或退款。
- 保修期限根据产品性质和价格而定，不受固定时间硬性限制。

## 4. 退货细节
- **物流损坏:** 如果订单在交付时损坏，请在 **3天内** 通知我们进行免费更换。
- **退货时限:** 退货必须在 **7天内** 完成，且邮费由买家承担。
- **商品状态:** 退回的商品必须处于可二次销售的状态。
- **预付款/定金:** 定金不予退还；取消预订时将扣除价值损失。

## 5. 手机接收要求
- **凭证要求:** 可能需要提供原始购买收据。
- **锁定限制:** 我们不接受有运营商锁或激活锁（如 iCloud 锁）的设备。
- **账号退出:** 提交前必须完全退出原有账号，并移除所有屏幕锁、密码及安全措施
